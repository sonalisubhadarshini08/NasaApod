# NASA APOD Viewer (Express.js)

A simple Node.js and Express application that allows users to view NASAâ€™s **Astronomy Picture of the Day (APOD)** by selecting a date.  
The app fetches data from the NASA APOD API and displays the image along with its explanation in a clean, responsive UI.

---

## Features

- Select any valid date to view APOD
- Fetches data from NASA APOD API
- Displays high-resolution image and explanation
- Clean card-based UI
- Responsive layout for desktop and mobile

---

## Tech Stack

- Node.js
- Express.js
- HTTPS module
- Body-Parser
- HTML & CSS (inline styling)

---

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/nasa-apod-viewer.git
   ```

2. Navigate to the project folder
    ```bash
    cd nasa-apod-viewer
    ```

3. Install dependencies
    ```bash
    npm install
    ```

4. Run the code
    ```bash
    npm run dev
    ```

5. Visit http://localhost:3000

## Usage

- Open the application in your browser
- Select a date using the date picker
- Click View Picture
- The Astronomy Picture of the Day and its explanation will be displayed
